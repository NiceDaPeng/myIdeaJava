四大名著

  • 三国演义

    捷克方面4月17日声称，俄罗斯情报部门与2014年兹林州弗尔贝季采村军火库爆炸事件有关。随后捷克政府宣布驱逐18名俄罗斯外交人员。 俄外交部4月18日宣布驱逐捷克驻俄罗斯的20名外交人员。捷克外长库尔哈内克对此提出异议， 同时要求被俄罗斯驱逐出境的捷克外交人员必须能在22日12点之前返回莫斯科，否则的话， 捷克将继续驱逐俄罗斯驻捷克外交人员，直到将其人数缩减到与捷克驻俄罗斯外交人员人数相一致。
